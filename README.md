## Overview

💡 This project is a **real-time Credit Card Fraud Detection System**  
that uses a trained **Random Forest model** to classify transactions into:

- ✅ **Normal Transactions**
- ⚠️ **Fraudulent Transactions**

🔥 Designed for:
- Real-time predictions
- High accuracy
- Clean UI experience

---

## Features

✨ Interactive Streamlit UI  
⚡ Instant prediction  
📊 Pre-trained ML model  
🔄 Scaler-based preprocessing  
🌐 Model loading via GitHub  
🎨 Smooth UX
